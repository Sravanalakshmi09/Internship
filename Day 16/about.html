About SignalSense

SignalSense is a Machine Learning based IoT Anomaly Detection System developed to identify abnormal sensor behaviour from IoT devices. It combines Flask, Python, SQLite and Machine Learning into a user-friendly dashboard.

Project Overview

SignalSense predicts whether an IoT device is operating normally or showing abnormal behaviour using Temperature, Humidity and Battery values. The application uses a trained Logistic Regression model to classify the incoming sensor values and displays the prediction instantly.


Objective

Detect abnormal IoT sensor readings automatically using Machine Learning.

IoT Monitoring

Monitor sensor behaviour and identify possible device failures.

Real-Time Prediction

Generate instant predictions using the trained ML model.


Technologies Used

Interactive effects and animations.


System Workflow


1️⃣ User Input

User enters Temperature, Humidity and Battery values.

2️⃣ Data Processing

The Flask backend receives the input and applies feature scaling.

3️⃣ Prediction

The Logistic Regression model predicts Normal or Anomaly.

4️⃣ Database

Prediction is stored in SQLite for future reference.
