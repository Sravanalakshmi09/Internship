Project Features


Machine Learning

Uses a trained Logistic Regression model to classify sensor readings as Normal or Anomaly.

SQLite Database

Stores every prediction so users can review previous results.

Real-time Detection

Quickly identifies abnormal sensor values that may indicate equipment faults.


System Workflow


Step 1: Enter the Device ID, Temperature, Humidity and Battery values.


Step 2: Flask sends the values to the Machine Learning model.


Step 3: The Standard Scaler normalizes the values.


Step 4: Logistic Regression predicts the device status.


Step 5: The result is displayed and saved to SQLite.
